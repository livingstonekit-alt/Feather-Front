# Third-Party Software and Services

FeatherFront uses the following external software, libraries, and services.
Review each project for license details and attribution requirements.

## Core analysis and media

- BirdNET Analyzer
  - Purpose: bird-call inference from audio segments
  - Reference: https://github.com/kahst/BirdNET-Analyzer

- FFmpeg
  - Purpose: stream/device capture and audio segmentation
  - Reference: https://ffmpeg.org/

## Weather and geocoding

- Open-Meteo Forecast API
  - Purpose: real-time weather data
  - Reference: https://open-meteo.com/

- Open-Meteo Geocoding API
  - Purpose: location/ZIP resolution to coordinates
  - Reference: https://open-meteo.com/en/docs/geocoding-api

## Typography

- Google Fonts: Bebas Neue
  - Reference: https://fonts.google.com/specimen/Bebas+Neue

- Google Fonts: Space Grotesk
  - Reference: https://fonts.google.com/specimen/Space+Grotesk
